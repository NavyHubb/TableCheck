# 0.개요
*테이블 예약 서비스*
매장에 방문하기 전에 미리 테이블을 예약할 수 있도록 돕는 서비스

기술스택: Spring, Java, MySQL

# 1. 사용자 별 기능

## 1. 1. 매니저
- [x] 회원가입(파트너)
- [x] 매장 등록
- [x] 예약 승인/거절

## 1. 2. 고객
- [x] 회원가입(고객)
- [x] 매장 검색
- [x] 매장 상세정보 조회
- [x] 매장 예약
- [x] 방문 코드 확인
- [x] 리뷰 작성

## 1. 3. 키오스크
- [x] 방문 확인
- [x] 방문 코드 인증
